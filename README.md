## 

A shell script to test the performance of a few DNS resolvers from your location.

## macOS

### Option 1: Run via Curl (Quick Start)
Execute the following command directly in your Terminal:
```
curl -s https://raw.githubusercontent.com/dazzah87/dnstest/main/dnsperftest.sh | bash
```
### Option 2: Run locally (Manual)<br/>
If you prefer to download and run the script yourself, follow these steps:<br/>
	1) [Download](https://github.com/dazzah87/dnstest/blob/main/dnsperftest.sh) the latest version of the `dnsperftest.sh` script.<br/>
  2) Navigate to the directory where the `dnsperftest.sh` script is located: `cd /path/to/directory`<br/>
  3) Make the script executable: `chmod +x dnsperftest.sh`<br/>
  4) Run the script: `./dnsperftest.sh`
